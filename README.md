# Spring Boot Backend Projects for CSE & IT Students

## Overview
This repository contains **5 beginner-friendly Spring Boot backend projects** designed for CSE and IT students.  
Each project includes CRUD APIs, PostgreSQL integration, and REST endpoints to build practical backend skills.

## Projects Included

1. **Student Course Registration System**  
   Manage students, courses, and registrations (many-to-many relationship).

2. **Employee Task Management System**  
   Create tasks, assign to employees, update status (Todo/Doing/Done).

3. **Online Book Store**  
   View books, add to cart, and place simple orders.

4. **Bug Tracking System**  
   Report bugs, assign priorities, and update statuses.

5. **Hostel/PG Management System**  
   Manage rooms, tenants, and rent payments.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
